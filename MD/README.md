# Galaxy 3D Generator

Three.js와 Vite로 제작된 멋진 인터랙티브 3D 은하 생성기입니다. 이 애플리케이션을 통해 사용자는 다양한 매개변수와 프리셋을 사용하여 자신만의 독특한 은하를 생성하고 커스터마이징할 수 있으며, 아름다운 커스텀 쉐이더와 후처리 효과로 렌더링됩니다.

## 🚀 기술 스택

*   **[Three.js](https://threejs.org/):** 은하를 렌더링하고 씬(Scene)을 처리하기 위한 강력한 3D 라이브러리입니다.
*   **[Vite](https://vitejs.dev/):** 쾌적한 개발 경험을 제공하는 빠른 빌드 도구 및 개발 서버입니다.
*   **[Lil-gui](https://lil-gui.georgealways.com/):** 실시간으로 매개변수를 조정할 수 있는 가벼운 플로팅 GUI입니다.
*   **GLSL (OpenGL Shading Language):** 고성능 입자 렌더링 및 애니메이션을 위한 커스텀 버텍스 및 프래그먼트 쉐이더입니다.

## ✨ 주요 기능

*   **인터랙티브 은하 생성:** 수천 개의 입자로 구성된 복잡한 나선 은하를 생성합니다.
*   **실시간 커스터마이징:** 제어 패널을 사용하여 은하의 모든 측면을 조정할 수 있습니다:
    *   **Count:** 별(입자)의 개수를 조정합니다.
    *   **Size:** 개별 별의 크기를 변경합니다.
    *   **Radius:** 은하의 전체 크기를 설정합니다.
    *   **Branches:** 나선 팔의 개수를 제어합니다.
    *   **Spin:** 나선 팔의 휘어짐 정도를 조정합니다.
    *   **Randomness:** 더 자연스러운 모습을 위해 불규칙성을 추가합니다.
    *   **Colors:** 은하의 내부 및 외부 색상을 커스터마이징합니다.
*   **아름다운 프리셋:** 다양한 은하 스타일로 즉시 전환할 수 있습니다:
    *   🌀 **Classic Spiral (클래식 나선)**
    *   💥 **Supernova (초신성)**
    *   👻 **Ghostly (유령)**
    *   🕳️ **Black Hole (블랙홀)**
    *   🌌 **Nebula (성운)**
*   **후처리 효과:** **UnrealBloomPass**를 사용하여 빛나는 영화 같은 연출을 더했습니다.
*   **궤도 제어 (Orbit Controls):** 완전히 상호작용 가능한 카메라 제어 (회전, 줌, 이동)를 지원합니다.
*   **반응형 디자인:** 모든 화면 크기에 맞춰 자동으로 조정됩니다.

## 🛠️ 설치 및 설정

1.  **저장소 복제 (Clone):**
    ```bash
    git clone <repository-url>
    ```

2.  **의존성 설치:**
    프로젝트 디렉토리로 이동하여 다음을 실행합니다:
    ```bash
    npm install
    ```

3.  **개발 서버 실행:**
    ```bash
    npm run dev
    ```
    터미널에 표시된 URL(보통 `http://localhost:5173`)을 브라우저에서 엽니다.

4.  **프로덕션 빌드:**
    ```bash
    npm run build
    ```

## 🎮 사용 방법

*   오른쪽 상단의 **제어 패널 (Control Panel)**을 사용하여 은하 매개변수를 조정합니다.
*   드롭다운 메뉴에서 **프리셋 (Preset)**을 선택하여 다양한 은하 유형을 확인합니다.
*   **왼쪽 클릭 + 드래그:** 화면 회전
*   **오른쪽 클릭 + 드래그:** 화면 이동 (Pan)
*   **스크롤:** 줌 인/아웃
